# Mobility and Economic Productivity Analysis

## Overview
Proyecto de análisis exploratorio en Python para estudiar la relación entre movilidad urbana y productividad económica. Incluyó limpieza de datos, análisis de correlaciones y visualización, con el objetivo de identificar patrones relevantes y traducirlos en hallazgos claros y útiles.

## Objective
Explorar la relación entre indicadores de movilidad urbana, como congestión y tiempos de traslado, y variables de productividad económica como el PIB per cápita.

## Tools
- Python
- Pandas
- Exploratory Data Analysis
- Data Visualization
- Correlation Analysis

## Process
- Limpieza y transformación de datos
- Exploración de variables clave
- Análisis de correlaciones
- Generación de visualizaciones
- Interpretación de hallazgos principales

## Key Findings
- Se observaron patrones relevantes entre tiempos de traslado y productividad
- La eficiencia de movilidad mostró relación con el desempeño económico
- Las visualizaciones ayudaron a comunicar tendencias de forma clara

## Business Value
Este proyecto demuestra cómo variables urbanas y operativas pueden convertirse en insights útiles para análisis económico y toma de decisiones.
